# 🛡️ CryptoKey Manager

**CryptoKey Manager** — это легковесное и безопасное десктопное приложение для управления паролями. Написано на **C++** с использованием современного графического стека **Qt 6 (QML)**. Программа реализует локальное хранилище с применением военных стандартов шифрования.

---

## 🔐 Особенности безопасности

Приложение разработано с учетом защиты от несанкционированного доступа к данным:

*   **Алгоритм AES-256 (CBC Mode):** Все пароли хранятся в зашифрованном виде. Даже если файл базы данных будет украден, прочитать пароли без мастер-ключа невозможно.
*   **Производный ключ (Key Derivation):** Мастер-пароль пользователя не хранится в базе. Он используется для генерации 256-битного ключа шифрования через хеш-функцию **SHA-256**.
*   **Локальное хранилище SQLite:** Никаких облаков и серверов. Ваши пароли принадлежат только вам и хранятся в зашифрованном файле `password.db` на вашем диске.
*   **Безопасная очистка сессии:** Ключ шифрования существует только в оперативной памяти во время работы программы и стирается при выходе.

## 🚀 Основной функционал

- [x] **Защищенный вход:** Доступ к базе только по мастер-паролю.
- [x] **Шифрование "на лету":** Данные шифруются в момент нажатия кнопки "Сохранить".
- [x] **Менеджмент записей:** Добавление, просмотр и удаление учетных данных.
- [x] **Дешифрование по требованию:** Просмотр пароля происходит только после ввода конкретного ID записи.
- [x] **Современный UI:** Темная тема интерфейса, оптимизированная для десктопа.

## 🛠 Технологический стек

*   **Framework:** Qt 6.8+ (Quick/QML)
*   **Language:** C++17
*   **Build System:** CMake 3.16+
*   **Database:** SQLite 3
*   **Encryption:** 
    *   [QAESEncryption](https://github.com/brimpl/quat_aes) (AES-256 implementation)
    *   `QCryptographicHash` (SHA-256)

## 📦 Сборка и запуск

### Предварительные требования
1. Установленный **Qt 6.8+** (с компонентами Quick и SQL).
2. Компилятор с поддержкой **C++17** (MinGW 11+, MSVC 2019+ или Clang).
3. **CMake** 3.16 или выше.
